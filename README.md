# Todo list app
A todo list app


### General steps to follow when creating a project

- Thinking about the UI
- Functionality
	- Add a TODO (Button as well as Enter Key)
	- Delete a TODO
	- Check task
	- Total items count
- Data
	- tasks - an array
	- task - {title, completed , id}
- Functions (in code)
	- addTodo
	- deleteTodo
	- checkTodo
	- renderTodosList
	- showNotification
